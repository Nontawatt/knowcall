# KnowCall

## แนวคิด (Concept)

KnowCall เป็นแอปพลิเคชันป้องกันและแจ้งเตือนสายโทรศัพท์พอร์ทโทรศัพท์หรือข้อความที่เป็นมิจฉาชีพุณและสแปมสำหรับฬำหรับผู้ใช้ในประเทศไทย โดยเน้นความปลอดภัยและความเป็นส่วนตัวของผู้ใช้ แอปนี้จะมีฐานข้อมูลเลขโทรศัพท์และโดเมนที่น่าสงสัยที่อัปเดตโดยชุมชนและหน่วยงานที่เกี่ยวข้อแม่นยำที่โกดมากยิ่งขึ้น

## คุณสมบัติสำคัญ

- บล็อกสายโทรศัพท์จากหมายเลขที่ซ่อน หรือหมายเลขที่ไม่ได้บันทึกในสมุดโทรศัพท์
- บล็อกสายจากหมายเลขต่างประเทศหรือหมายเลขที่อยู่ในรายการที่ผู้ใช้ไม่ต้องการรับ
- จัดการรายการขาว (Whitelist) และรายการบล็อก (Blacklist) เพื่อควบคุ้มการรับสาย
- ตรวจหมายเลขโทรศัพท์ทองหลายแหล่งข้อมูล เช่น UnknownPhone, Tellows และ TrueCaller เพื่อความแม่นยำโดยอ้างทริงโฉติพระตูแอปซ็อนข้อมูล
- รองรับมาตรฐาน STIR/SHAKEN เพื่อยืนยัน Caller ID และลดการปลอมแปลงสาย
- โมดปิดเสียง (Auto‑Mute) สำหรับสายที่ตรวจพบว่าเป็นสแปมหรือโฆษณา
- ระบบรายงานหมายเลข: ผู้ใช้สามารถารส่งหมายเลขที่ต้องสงสัยหรือยืนยันว่าเป็นหมายเลขปลอดภัยเพื่ออัปเดต
- อัปเดตฐานถานข้อมูลอัตโมติด้วยข้อมูลจากหน่วยงานกำกับดูและจะลวงไม่แบ่งพอดที่ได้รับอนุญาติตัว
- การแจ้งเตือนพิเศษฺสำหรับสายต่างประเทศตามข้อกำหนดของ กสทช. และ PDPC และระบบยืนยันชื่อบัญชีสำ฻รับการโอนเงินเพื่อป้องกันบัญชีม้า
- อินเทอร์เฟซภาษาไทยที่ใช้งายและใช้งาน พร้อมระบบช่วยเหลือการหลีกเลี่ยงกลโกงใหม่
